D1 export
After exporting, run these in a terminal with Wrangler installed and authenticated. Replace <YOUR_DB_NAME> with your D1 database name.
Single file
wrangler d1 execute <YOUR_DB_NAME> --remote --file ".\youtube-watched-history-d1.sql"
Multiple chunk files
Get-ChildItem ".\youtube-watched-history-d1-part-*.sql" |
    Sort-Object Name |
    ForEach-Object {
        wrangler d1 execute <YOUR_DB_NAME> --remote --file $_.FullName
    }
Cloudflare sync sends only pending unsynced data to reduce API calls. The API token and auto sync cadence are shared across history and subscription sync workers.
